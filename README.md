# ML Demand Forecasting Platform

An end-to-end ML platform that predicts product demand, detects sales trends,
identifies anomalies, and surfaces business insights from historical retail data.

## Problem Statement
Retail and e-commerce businesses need to anticipate product demand to manage
inventory, pricing, and promotions. This project builds a system that ingests
historical sales data, trains and compares forecasting models, and exposes
predictions and insights through a REST API and dashboard.

## Tech Stack
- **Backend:** Python, FastAPI
- **ML:** scikit-learn, XGBoost, LightGBM, Pandas, NumPy
- **Visualization:** Matplotlib, Plotly
- **Frontend:** React.js
- **Deployment:** Docker

## Setup
```bash
# backend
cd api
pip install -r requirements.txt
uvicorn app.main:app --reload

# generate sample dataset
python data/generate_dataset.py
```

API docs available at `http://localhost:8000/docs` (Swagger, auto-generated by FastAPI).

## Dataset
`data/sales_dataset.csv` — synthetic daily sales across 5 stores × 8 products,
with trend, weekly/yearly seasonality, promo effects, injected anomalies, and
intentional missing values/duplicates for cleaning practice.
